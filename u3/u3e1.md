# Tema 1. Uso avanzado de funciones

## Unidad 3. Promesas

## Ejercicio 1.3.1 (Castellano)

Implementa una función denominada `getAJoke` que devuelve la promesa resultante de realizar una petición mediante el uso de `fetch` al siguiente endpoint:

```
https://geek-jokes.sameerkumar.website/api?format=json
```

Para [conocer más sobre el endpoint en este enlace](https://github.com/sameerkumar18/geek-joke-api).

La función que debes implementar ha de recibir un único parámetro denominado `callback` que será una función a la que se ha de llamar en caso de error, como se describe a continuación partiendo de esa petición `fetch`:

- Si se recibe respuesta se ha de procesar resolviendo la promesa con el valor JSON de la respuesta.

- Si se produce un error se ha de llamar al **callback** recibido como parámetro pasándole al mismo el error capturado desde la petición `fetch`.
